# Desarollo-de-Software

En este proyecto podremos ir visualizando el avance de programación de nuestro juego Piedra, papel o tijera, en el cual se estará explicando a travez de diagramas de flujo y codificando por medio de Python.

# Reglas

- Tijera gana a papel
- Papel gana a piedra
- Si ambos eligen lo mismo, es empate
